          <p><i>Aprende a configurar tu cuenta de Cloudflare y a activar tu dominio con unos tutoriales de vídeos breves.</i></p>
          <div class="cf-toc">
           <ul> 
            <li><a href="#1Al8BL3K2cq56r0baGDJ4u">Información general</a></li> 
            <li><a href="#24ZFznWWFk3vwaxeMGmlFc">Cómo agregar un dominio a Cloudflare</a></li>

           </ul>
          </div>
          <hr>
          <h2 id="1Al8BL3K2cq56r0baGDJ4u">Información general</h2>
          <p>Empezar a trabajar con Cloudflare es un procesos de varios pasos. Los siguientes tutoriales de vídeo te guiarán a través del proceso de incorporación y podrás descubrir algunas de las prácticas recomendadas.</p>
          <hr>
          <h2 id="24ZFznWWFk3vwaxeMGmlFc">Cómo agregar un dominio a Cloudflare</h2>
          <p>Cuando creas una nueva cuenta de Cloudflare, parte del proceso incluye <u>añadir un nuevo dominio</u>.</p>
          <p></p>
          <p class="cf-video-container"> <iframe style="border:none" src="https://watch.cloudflarestream.com/1434cf5af367af0385e4c54d8300ea59" width="480" height="320" allowfullscreen></iframe> </p> 
          <hr>
          <h2 id="66zv5PsLZLwoJRnOlssKPK">Cambiar los servidores de nombres de dominio a Cloudflare</h2>
          <p>Para utilizar Cloudflare como proveedor de DNS autoritativo, tienes que <a href="https://support.cloudflare.com/hc/es-es/articles/205195708-Changing-your-domain-nameservers-to-Cloudflare"><u>actualizar los servidores de nombres</u></a> en tu registrador de dominio para que el tráfico web se enrute a través de la red de Cloudflare. Este vídeo explica cómo cambiar tus servidores de nombres en tu registrador de dominios cuando añadas tu sitio web a Cloudflare.</p>
          <p></p>
          <p class="cf-video-container"> <iframe style="border:none" src="https://watch.cloudflarestream.com/d29bdc08f20edcb9f7ec8223f893bb21" width="480" height="320" allowfullscreen></iframe> </p>
          <hr>
          <h2 id="4UFBdCRZNniEV2xsnQHpqt">Añadir registros MX a Cloudflare</h2>
          <p>Es necesario añadir registros DNS a Cloudflare durante el proceso de incorporación. El DNS de Cloudflare solo redirecciona mediante proxy tráfico HTTP, por lo que los registros MX no se pueden redireccionar mediante proxy a través de nuestra red. Este vídeo muestra las prácticas recomendadas durante el proceso de <a href="https://support.cloudflare.com/hc/es-es/articles/200168876-Email-undeliverable-when-using-Cloudflare"><u>añadir registros MX a Cloudflare</u></a>.</p>
          <p></p>
          <p class="cf-video-container"> <iframe style="border:none" src="https://watch.cloudflarestream.com/787115d7926481486d49e361d25e9cf6" width="480" height="320" allowfullscreen></iframe> </p>
          <hr>
          <h2 id="6dEVd998jdo06Kv0kZUYiK">Descripción del funcionamiento</h2>
          <p>Cloudflare crea funciones que mejoran el rendimiento de tu sitio. Este vídeo describe características clave para mejorar el funcionamiento, incluidos Auto-Minify, Brotli, Polish y WebP.</p>
          <p></p>
          <div class="cf-sidebar-2 cf-note">
           La activación de algunas de estas funciones requiere un plan Pro. 
          </div>
          <p></p>
          <p class="cf-video-container"> <iframe style="border:none" src="https://watch.cloudflarestream.com/33c5906bb4c591856b66f66db2df6019" width="480" height="320" allowfullscreen></iframe> </p>
          <hr>
          <h2 id="60TYAhsj31sVZZV4FlPa5j">Descripción general de seguridad</h2>
          <p>Cloudflare ofrece muchas soluciones para encriptar el tráfico web. Este vídeo describe los conceptos básicos de HTTP y los HTTP, y muestra cómo asegurar y proteger tu sitio con la pestaña SSL/TLS de Cloudflare.&nbsp;</p>
          <p></p>
          <p class="cf-video-container"> <iframe style="border:none" src="https://watch.cloudflarestream.com/d9ed966c74c7b367db155b5c01f6b56c" width="480" height="320" allowfullscreen></iframe> </p>
          <p></p>
