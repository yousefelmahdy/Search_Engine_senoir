function fix_stickyad() {

  var elem = document.getElementById("stickyadcontainer");
  if (!elem) {return false;}
  if (document.getElementById("skyscraper")) {
    var skyWidth = Number(w3_getStyleValue(document.getElementById("skyscraper"), "width").replace("px", ""));  
    }
  else {
    var skyWidth = Number(w3_getStyleValue(document.getElementById("right"), "width").replace("px", ""));  
  }
  elem.style.width = skyWidth + "px";
  if (window.innerWidth <= 992) {
    elem.style.position = "";
    elem.style.top = stickypos + "px";
    return false;
  }
  var stickypos = document.getElementById("stickypos").offsetTop;
  var docTop = window.pageYOffset || document.documentElement.scrollTop || document.body.scrollTop;
  var adHeight = Number(w3_getStyleValue(elem, "height").replace("px", ""));
  if (stickyadstatus == "") {
    if ((stickypos - docTop) < 60) {
      elem.style.position = "fixed";
      elem.style.top = "60px";
      stickyadstatus = "sticky";
      document.getElementById("stickypos").style.position = "sticky";

    }
  } else {
    if ((docTop + 60) - stickypos < 0) {  
      elem.style.position = "";
      elem.style.top = stickypos + "px";
      stickyadstatus = "";
      document.getElementById("stickypos").style.position = "static";
    }
  }
  if (stickyadstatus == "sticky") {
    if ((docTop + adHeight + 60) > document.getElementById("footer").offsetTop) {
      elem.style.position = "absolute";
      elem.style.top = (document.getElementById("footer").offsetTop - adHeight) + "px";
      document.getElementById("stickypos").style.position = "static";
    } else {
        elem.style.position = "fixed";
        elem.style.top = "60px";
        stickyadstatus = "sticky";
        document.getElementById("stickypos").style.position = "sticky";
    }
  }
}

       <div class="w3-cell w3-container w3-light-gray w3-border-left w3-cell-top"> 
        <p>JSON stands for <b>J</b>ava<b>S</b>cript <b>O</b>bject <b>N</b>otation</p>

     <h2>JSON Example</h2> 
     <p>This example defines an object:</p> 
     <div class="w3-example"> 
      <div class="w3-code notranslate jsHigh">
        {"name":"John", "age":30, "car":null} 
      </div> 
     </div> 
     <p>The person object has 3 properties:</p> 
     <ul> 
      <li>name</li> 
      <li>age</li> 
      <li>car</li> 
     </ul> 
     <p>Each property has a value.</p> 
     <hr> 
     <h2>What is JSON?</h2> 
     <ul> 
      <li>JSON stands for <b>J</b>ava<b>S</b>cript <b>O</b>bject <b>N</b>otation</li> 
      <li>JSON is text, written with JavaScript object notation</li> 
      <li>JSON is a lightweight data-interchange format</li> 
      <li>JSON is used to send data between computers</li> 
      <li>JSON is language independent <b>*</b></li> 
     </ul> 
     <div class="w3-panel w3-note"> 
      <p>*<br> The JSON syntax is derived from JavaScript object notation, but the JSON format is text only.</p> 
      <p>Code for reading and generating JSON exists in many programming languages.</p> 
     </div> 
     <p>The JSON format was originally specified by <a target="_blank" href="http://www.crockford.com">Douglas Crockford</a>.</p> 
     <hr> 
     <h2>Why Use JSON?</h2> 
     <p>The JSON format is syntactically similar to the code for creating JavaScript objects.</p> 
     <p>Because of this similarity, a JavaScript program can easily convert JSON data into JavaScript objects.</p> 
     <p>Since the JSON format is text only, it can easily be sent to and from a server, and used as a data format by any programming language.</p> 
     <p>JavaScript has a built in function for converting JSON strings into native JavaScript objects:</p> 
     <p><code class="w3-codespan">JSON.parse()</code></p> 
     <p>So, if you receive data from a server, in JSON format, you can use it like any other JavaScript object.</p> 
     <p>JavaScript also has a built in function for converting an object into a JSON string:</p> 
     <p><code class="w3-codespan">JSON.stringify()</code></p> 
     <p>So, you can send JavaScript objects to a server in pure text format.</p>

     <h2>Exchanging Data</h2> 
     <p>When exchanging data between a browser and a server, the data can only be text.</p> 
     <p>JSON is text, so you can convert any JavaScript object into JSON, and send the JSON text to a server.</p> 
     <p>You can also convert any JSON text received from the server into JavaScript objects in your programs.</p> 
     <p>This way you can work with data as JavaScript objects, with no complicated parsing and translations.</p> 
     <hr> 
     <h2>Storing Data</h2> 
     <p>When storing data, the data has to be a certain format, and regardless of where you choose to store it, <em>text</em> is always one of the legal formats.</p> 
     <p>JSON makes it possible to store JavaScript objects as text.</p>
