   <section id="faq" class="section" aria-label="Frequently asked questions about Azure Kubernetes Service (AKS)" data-module="faq-01"> 
    <div class="row"> 
     <div class="column medium-3"> 
      <h2>Frequently asked questions about Azure Kubernetes Service (AKS)</h2> 
      <div class="row row-size4 column"> 
       <ul class="list--links"> 
        <li> <a href="/en-us/global-infrastructure/services/?products=kubernetes-service" data-bi-slot="1 ">Check availability by region</a> </li> 
        <li> <a href="https://status.azure.com/" data-bi-slot="2 ">View current operational status</a> </li> 
        <li> <a href="/en-us/support/legal/sla/kubernetes-service/" data-bi-slot="3 ">Read the service level agreement (SLA)</a> </li> 
       </ul> 
      </div> 
     </div> 
     <div class="column medium-8 medium-offset-1"> 
      <ul class="faq"> 
       <li class="faq__item"> <button type="button" class="faq__question" aria-expanded="false" aria-controls="0eddbd73" data-bi-slot="1" data-bi-id="faq-1" data-bi-an="faq-01"> Where is AKS available? </button> 
        <div class="faq__answer" id="0eddbd73">
         <p>AKS is available in more than <a href="/en-us/global-infrastructure/services/?products=kubernetes-service" data-event="global-sd-unnamed-clicked-faqlink">42 regions worldwide</a>, and new ones are being added regularly.</p>
        </div> </li> 
       <li class="faq__item"> <button type="button" class="faq__question" aria-expanded="false" aria-controls="5fc79cd1" data-bi-slot="2" data-bi-id="faq-2" data-bi-an="faq-01"> What is Kubernetes? </button> 
        <div class="faq__answer" id="5fc79cd1">
         <p>Kubernetes is open-source software that helps deploy and manage containerized applications at scale. It orchestrates a cluster of Azure virtual machines, schedules containers, automatically manages service discovery, incorporates load balancing, and tracks resource allocation. It also checks the health of individual resources and heals apps with auto-restart and auto-replication.</p> 
         <p>AKS provides a managed Kubernetes service with automated provisioning, upgrading, monitoring, and on-demand scaling. To learn more, visit the Kubernetes <a href="https://kubernetes.io/" data-event="global-sd-unnamed-clicked-faqlink">website</a> and the <a href="/en-us/topic/what-is-kubernetes/" data-event="global-sd-unnamed-clicked-faqlink">topic page</a>.</p>
        </div> </li> 
       <li class="faq__item"> <button type="button" class="faq__question" aria-expanded="false" aria-controls="44836bc2" data-bi-slot="3" data-bi-id="faq-3" data-bi-an="faq-01"> Whats the difference between Docker and Kubernetes? </button> 
        <div class="faq__answer" id="44836bc2">
         <p>Docker and Kubernetes are complementary. Docker provides an open standard for packaging and distributing containerized applications, while Kubernetes provides for the orchestration and management of distributed, containerized applications created with Docker. In other words, Kubernetes provides the infrastructure needed to deploy and run applications built with Docker. </p>
        </div> </li> 
      </ul>
