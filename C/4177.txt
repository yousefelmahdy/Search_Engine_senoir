       <p><small></small> </p> 
       <div class="templatebox-desc"> 
        <p>{{edit request}} is to be typed on the talk page. Please make the request as clear and specific as possible and include a reason for the change. This is to be used for non-controversial changes or when there is a clear consensus for the changes. </p>
        <p>The page on which the request is placed is automatically categorised into <a href="/wiki/Category:Commons_protected_edit_requests" title="Category:Commons protected edit requests">Category:Commons protected edit requests</a>. Administrators will check this category for protected edit requests to process. </p>
        <p>On Mediawiki messages' talk pages, you can add <a href="/wiki/Template:MWheader" title="Template:MWheader">{{MWheader}}</a> to help other users become familiar with this procedure. </p>
        <p>On Templates' talk pages, you can add <a href="/wiki/Template:Permprot" title="Template:Permprot">{{Permprot}}</a> to help other users become familiar with this procedure. </p> 
       </div> 
       <p>The template automatically sets the following categories: <a href="/wiki/Category:Commons_protected_edit_requests" title="Category:Commons protected edit requests">Category:Commons protected edit requests</a> </p>
