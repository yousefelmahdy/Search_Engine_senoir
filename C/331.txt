       <h1 class="twtr-type--page-headline">About twitter.com supported browsers</h1> 
       <div class="c01-rich-text-editor twtr-component-space--md twtr-rte"> 
        <p>We officially support versions of the following browsers from the past year on twitter.com:</p> 
        <ul>
         <li>Edge (<a href="https://microsoft.com/edge" target="_blank" rel="nofollow noopener noreferrer">https://microsoft.com/edge</a>)</li>
         <li>Safari (<a href="https://www.apple.com/safari" target="_blank" rel="nofollow noopener noreferrer">https://www.apple.com/safari</a>)</li>
         <li>Chrome (<a href="https://www.google.com/chrome" target="_blank" rel="nofollow noopener noreferrer">https://www.google.com/chrome</a>)</li>
         <li>Firefox (<a href="https://www.mozilla.org/firefox" target="_blank" rel="nofollow noopener noreferrer">https://www.mozilla.org/firefox</a>)</li>
         <li>Any browser based on Webkit / Chromium engines (Opera, Samsung Internet, UCBrowser, etc.)</li>
        </ul> 
        <p>If you use Twitter on an older version of the above browsers, browsers not listed, or a browser extension which modifies your user agent, you may be unable to use Twitter. In order to ensure the best possible Twitter experience, including all the latest features and security updates, please make sure your browser and operating system are up-to-date.<br> </p> 
        <p><br> Please note that compatible browsers are included with most operating systems:<br> </p> 
        <ul>
         <li>Windows 10 - Edge</li>
         <li>Android - Chrome</li>
         <li>MacOS / iOS - Safari</li>
        </ul>
