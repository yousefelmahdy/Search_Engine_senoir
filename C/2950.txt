       <h3 id="introduction"><a href="#introduction">Introduction</a></h3> 
       <p>Educators who teach a course on software development can use discounts, partnerships, training, and tools from GitHub Education to effectively teach relevant skills to students.</p> 
       <p>In this guide, you'll get started with GitHub, sign up for accounts and discounted services through GitHub Education, and create a space for your course and assignment on GitHub Classroom.</p> 
       <div class="extended-markdown tip border rounded-1 mb-4 p-3 color-border-info color-bg-info f5"> 
        <p><strong>Tip</strong>: If you're a student and you'd like to take advantage of an academic discount, see "<a href="/en/github/teaching-and-learning-with-github-education/applying-for-a-student-developer-pack">Applying for a student developer pack</a>."</p> 
       </div> 
       <h3 id="creating-accounts-on-github"><a href="#creating-accounts-on-github">Creating accounts on GitHub</a></h3> 
       <p>First, you'll need to create a free user account on GitHub.</p> 
       <ol> 
        <li>Go to GitHub's <a href="https://github.com/pricing">Pricing</a> page.</li> 
        <li>Read the information about the different products and subscriptions that GitHub offers, then click the upgrade button under the subscription you'd like to choose.</li> 
        <li>Follow the prompts to create your free user account.</li> 
       </ol> 
       <p>After you create your user account, create a free organization account. You'll use this organization account to create and manage classrooms GitHub Classroom.</p> 
       <ol> 
        <li>In the upper-right corner of any page, click your profile photo, then click <strong>Settings</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/userbar-account-settings.png" alt="Settings icon in the user bar"></span></li> 
        <li>In your user settings sidebar, click <strong>Organizations</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/settings-user-orgs.png" alt="User settings for organizations"></span></li> 
        <li>In the "Organizations" section, click <strong>New organization</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/settings/new-org-button.png" alt="Button for creating a new organization"></span></li> 
        <li>Follow the prompts to create a free organization.</li> 
       </ol> 
       <p>For more information, see "<a href="/en/github/getting-started-with-github/types-of-github-accounts">Types of GitHub accounts</a>."</p> 
       <h3 id="applying-for-an-educator-discount"><a href="#applying-for-an-educator-discount">Applying for an educator discount</a></h3> 
       <p>Next, you'll sign up for discounts on services from GitHub. To apply for an educator or researcher discount, you must meet the following requirements.</p> 
       <ul> 
        <li>Be an educator, faculty member, or researcher.</li> 
        <li>Have a verifiable school-issued email address.</li> 
        <li>Upload documents that prove your school affiliation.</li> 
        <li>Have a user account on GitHub.</li> 
       </ul> 
       <div class="extended-markdown tip border rounded-1 mb-4 p-3 color-border-info color-bg-info f5"> 
        <p><strong>Tip</strong> In addition to individual discounts, GitHub offers partnerships with educational institutions through the GitHub Campus Program. For more information, see the <a href="https://education.github.com/schools">GitHub Campus Program</a> website.</p> 
       </div> 
       <ol> 
        <li> <p>Go to <a href="https://education.github.com">GitHub Education</a> and, in the top right, click <strong>Get benefits</strong>.</p> </li> 
        <li> <p>Under "Individuals", click <strong>Get teacher benefits</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/education/click-get-teacher-benefits.png" alt="&quot;Get teacher benefits&quot; on GitHub Education homepage"></span></p> </li> 
        <li> <p>Select or add the email address you use for school. <span class="procedural-image-wrapper"><img src="/assets/images/help/education/select-email-address.png" alt="Select the email address"></span></p> 
         <div class="extended-markdown tip border rounded-1 mb-4 p-3 color-border-info color-bg-info f5"> 
          <p> <strong>Tip</strong>: Selecting a school-issued email address, if you have one, gives you the best chance of a speedy review.</p> 
         </div> </li> 
        <li> <p>If prompted, upload proof of your academic status.</p> 
         <ul> 
          <li> <p>Click <strong>Use your camera to take a picture</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/education/upload-proof-status.png" alt="Button to use your camera to take a picture"></span></p> </li> 
          <li> <p>Optionally, to change the camera you want to use, use the camera drop-down menu. <span class="procedural-image-wrapper"><img src="/assets/images/help/education/camera-drop-down.png" alt="Camera drop-down menu"></span></p> </li> 
          <li> <p>Place your valid academic ID or other proof of current academic status in the frame, then click <strong>Take photo</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/education/take-photo-button.png" alt="Take photo button"></span></p> </li> 
          <li> <p>If you can't take a picture of your proof of academic status, to upload an image from your computer instead, click <strong>Upload image</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/education/upload-image-link.png" alt="Upload image link"></span></p> 
           <div class="extended-markdown note border rounded-1 mb-4 p-3 color-border-info color-bg-info f5"> 
            <p><strong>Note:</strong> PDF files are not supported, and you cannot upload them as proof of academic status.</p> 
           </div> </li> 
         </ul> </li> 
        <li> <p>Enter your school's name. <span class="procedural-image-wrapper"><img src="/assets/images/help/education/enter-school-name.png" alt="Field for your school name"></span></p> </li> 
        <li> <p>Describe how you plan to use GitHub. <span class="procedural-image-wrapper"><img src="/assets/images/help/education/purpose-for-github-education.png" alt="Description field for how you plan to use GitHub"></span></p> </li> 
        <li> <p>Verify your application details, then click <strong>Submit your information</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/education/submit-your-information-button.png" alt="Submit your information button"></span></p> <p>If your application is approved, you'll receive a confirmation email. Applications are usually processed within a few days, but it may take longer during peak times, such as during the start of a new semester.</p> </li> 
       </ol> 
       <h3 id="set-up-github-classroom"><a href="#set-up-github-classroom">Set up GitHub Classroom</a></h3> 
       <p>With your user account and organization account, you're ready to get started with GitHub Classroom. GitHub Classroom is free to use. You can track and manage assignments, grade work automatically, and provide feedback to your students.</p> 
       <ol> 
        <li> <p>Sign into <a href="https://classroom.github.com/login">GitHub Classroom</a>.</p> </li> 
        <li> <p>To authorize GitHub Classroom to access your user account on GitHub, review the information, then click <strong>Authorize GitHub Classroom</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/classroom/setup-click-authorize-github-classroom.png" alt="&quot;Authorize GitHub Classroom&quot; button for user account"></span></p> </li> 
        <li> <p>Review the information. To authorize GitHub Classroom to access your organization account on GitHub, click <strong>Grant</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/classroom/setup-click-grant.png" alt="&quot;Grant&quot; button for organization"></span></p> 
         <div class="extended-markdown tip border rounded-1 mb-4 p-3 color-border-info color-bg-info f5"> 
          <p><strong>Tip</strong>: If you see a <strong>Request</strong> button instead of a <strong>Grant</strong> button, you are a member of the organization, not an owner. An owner must approve your request for GitHub Classroom. You must be an organization owner to create and manage classrooms and assignments in GitHub Classroom. For more information, see "<a href="/en/github/authenticating-to-github/authorizing-oauth-apps#oauth-apps-and-organizations">Authorizing OAuth Apps</a>."</p> 
         </div> </li> 
        <li> <p>Click <strong>Authorize github</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/classroom/setup-click-authorize-github.png" alt="Click &quot;Authorize&quot; button for organization"></span></p> </li> 
       </ol> 
       <h3 id="create-your-classroom"><a href="#create-your-classroom">Create your classroom</a></h3> 
       <p>A classroom in GitHub Classroom is where teachers and students interact during a course. Teachers can create a student roster for the classroom, then create, assign, review, and grade assignments within the classroom.</p> 
       <ol> 
        <li>Sign into <a href="https://classroom.github.com/login">GitHub Classroom</a>.</li> 
        <li>Click <strong>Create your first classroom</strong> or <strong>New classroom</strong>.</li> 
        <li>In the list of organizations, click the organization you'd like to use for your classroom. Optionally, you can create a new organization. For more information, see "<a href="/en/organizations/collaborating-with-groups-in-organizations/creating-a-new-organization-from-scratch">Creating a new organization from scratch</a>." <span class="procedural-image-wrapper"><img src="/assets/images/help/classroom/click-organization.png" alt="Organization in list of organizations for creating new classroom"></span></li> 
        <li>Type the name for your classroom. <span class="procedural-image-wrapper"><img src="/assets/images/help/classroom/type-classroom-name.png" alt="Text field for typing name of classroom"></span></li> 
        <li>Click <strong>Create classroom</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/classroom/click-create-classroom-button.png" alt="Text field for typing name of classroom"></span></li> 
       </ol> 
       <h3 id="next-steps"><a href="#next-steps">Next steps</a></h3> 
       <p>You've created a classroom, and you're ready to enrich your course with GitHub and GitHub Classroom! ?</p> 
       <ul> 
        <li>Watch some videos about GitHub Classroom. For more information, see "<a href="/en/education/manage-coursework-with-github-classroom/basics-of-setting-up-github-classroom">Basics of setting up GitHub Classroom</a>."</li> 
        <li>Manage your classrooms and classroom admins, and create a roster of students for your classroom. For more information, see "<a href="/en/education/manage-coursework-with-github-classroom/manage-classrooms">Manage classrooms</a>."</li> 
        <li>Use the Git and GitHub starter assignment to give students an overview of Git and GitHub fundamentals. For more information, see "<a href="/en/education/manage-coursework-with-github-classroom/use-the-git-and-github-starter-assignment">Use the Git and GitHub starter assignment</a>."</li> 
        <li>Create an assignment for individual students or teams. For more information, see "<a href="/en/education/manage-coursework-with-github-classroom/create-an-individual-assignment">Create an individual assignment</a>" or "<a href="/en/education/manage-coursework-with-github-classroom/create-a-group-assignment">Create a group assignment</a>."</li> 
        <li>Write and implement automated tests to provide immediate feedback to students directly in assignment repositories. For more information, see "<a href="/en/education/manage-coursework-with-github-classroom/use-autograding">Use autograding</a>."</li> 
        <li>Participate in the <a href="https://education.github.community">GitHub Education Community</a>.</li> 
       </ul>
