      <p>Q1: The <a href="/wiki/Wikipedia:DYKTN#hook" class="mw-redirect" title="Wikipedia:DYKTN">hook</a> itself should be <b>concise</b> (fewer than about 200 <a href="/wiki/Wikipedia:DYKTN#characters" class="mw-redirect" title="Wikipedia:DYKTN">characters</a>, including spaces and the question mark, but excluding the "..." or any "(pictured)"). While 200 is an outside limit, hooks slightly under 200 characters may still be rejected at the discretion of the selecting reviewers and administrators. <a href="/wiki/Wikipedia:Did_you_know/Multiple_articles" title="Wikipedia:Did you know/Multiple articles">Multiple articles are an exception.</a> You can count the 200 characters manually. Otherwise, select the text as it displays to the public (not from the edit screen which contains <a href="/wiki/Wikipedia:Wikitext" class="mw-redirect" title="Wikipedia:Wikitext">wikitext</a>), and then <a href="/wiki/Cut,_copy_and_paste#Copy_and_paste" class="mw-redirect" title="Cut, copy and paste">copy and paste</a> it into a blank document using a free website like <a rel="nofollow" class="external text" href="http://www.javascriptkit.com/script/script2/charcount.shtml">this</a>, or using an external <a href="/wiki/Software" title="Software">software</a> <a href="/wiki/Computer_program" title="Computer program">program</a> like <a href="/wiki/Microsoft_Word" title="Microsoft Word">Microsoft Word</a> that has a character-counting feature. Don't use the character count on the <a href="/wiki/T:TDYK" class="mw-redirect" title="T:TDYK">suggestion page</a> history screen, which includes signatures and other comments. </p> <!--
