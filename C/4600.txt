        <h2><span class="mw-headline" id="Instructions">Instructions</span></h2> 
        <p>Example: To redirect from <a href="/wiki/Category:Authors" title="Category:Authors">Category:Authors</a> to <a href="/wiki/Category:Writers" title="Category:Writers">Category:Writers</a>, simply add the following to <a href="/wiki/Category:Authors" title="Category:Authors">Category:Authors</a>: </p> 
        <dl>
         <dd>
          <code>{{Category redirect|Writers}}</code> or
         </dd> 
         <dd>
          <code>{{Category redirect|Category:Writers}}</code> with optional namespace prefix
         </dd>
        </dl> 
        <p>This template adds category pages to <a href="/wiki/Category:Wikipedia_soft_redirected_categories" title="Category:Wikipedia soft redirected categories">Category:Wikipedia soft redirected categories</a>. </p> 
        <h2><span class="mw-headline" id="Exceptions">Exceptions</span></h2> 
        <ul>
         <li>Non-empty categories will be listed at <a href="/wiki/Category:Wikipedia_non-empty_soft_redirected_categories" title="Category:Wikipedia non-empty soft redirected categories">Category:Wikipedia non-empty soft redirected categories</a><small> (?31?)</small>.</li> 
         <li>Infinite loops of length 1 will be listed at <a href="/wiki/Category:Wikipedia_category-redirect_box_parameter_needs_fixing" title="Category:Wikipedia category-redirect box parameter needs fixing">Category:Wikipedia category-redirect box parameter needs fixing</a><small> (?0?)</small> under <a href="/wiki/%E2%88%9E" class="mw-redirect" title="?">?</a>.</li> 
         <li>Redirects with non-existent targets will be listed at <a href="/wiki/Category:Wikipedia_category-redirect_box_parameter_needs_fixing" title="Category:Wikipedia category-redirect box parameter needs fixing">Category:Wikipedia category-redirect box parameter needs fixing</a><small> (?0?)</small> under <a href="/wiki/%E2%88%83" class="mw-redirect" title="?">?</a>.</li>
        </ul>
