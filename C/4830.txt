      <p>The <b>Help namespace</b> is a <a href="/wiki/Wikipedia:Namespace" title="Wikipedia:Namespace">namespace</a> consisting of <a href="/wiki/Wikipedia:HOWTOPAGES" class="mw-redirect" title="Wikipedia:HOWTOPAGES"> "how-to" and information pages</a> whose titles begin with the prefix <b>Help:</b>, such as <a href="/wiki/Help:Link" title="Help:Link">Help:Link</a>. </p>
      <p>These pages contain information intended to help use Wikipedia or its software. Some of these pages are intended for readers of the encyclopedia; others are intended for editors, whether beginning or advanced. Some of the pages in the Help namespace are copied from <a href="/wiki/Meta-Wiki" class="mw-redirect" title="Meta-Wiki">Meta-Wiki</a>. </p>
      <p>There is a large amount of overlap between the Help namespace and the <a href="/wiki/Wikipedia:Project_namespace" title="Wikipedia:Project namespace">Project namespace</a> (pages with the <i>Wikipedia:</i> prefix). For this reason <a href="/wiki/Help:Redirect" title="Help:Redirect">redirects</a> and <a href="/wiki/Wikipedia:Hatnote" title="Wikipedia:Hatnote">hatnotes</a> are often set up between these two namespaces. See <a href="/wiki/Help:About_the_help_pages" class="mw-redirect" title="Help:About the help pages">Help:About the help pages</a> for more information. </p>
