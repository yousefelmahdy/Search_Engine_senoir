     <hr> 
     <p class="intro">ASCII was the first character set (encoding standard) used between computers on the Internet.</p> 
     <p class="intro">Both ISO-8859-1 (default in HTML 4.01) and UTF-8 (default in HTML5), are built on ASCII.</p> 
     <hr> 
     <h2>The ASCII Character Set</h2> 
     <p>ASCII stands for the "American Standard Code for Information Interchange".</p> 
     <p>It was designed in the early 60's, as a standard character set for computers and electronic devices.</p> 
     <p>ASCII is a 7-bit character set containing 128 characters.</p> 
     <p>It contains the numbers from 0-9, the upper and lower case English letters from A to Z, and some special characters.</p> 
     <p>The character sets used in modern computers, in HTML, and on the Internet, are all based on ASCII.</p> 
     <p>The following tables list the 128 ASCII characters and their equivalent number.</p>

     <h2>ASCII Device Control Characters</h2> 
     <p>The ASCII control characters (range 00-31, plus 127) were designed to control hardware devices.</p> 
     <p>Control characters (except horizontal tab, line feed, and carriage return) have nothing to do inside an HTML document.</p>
