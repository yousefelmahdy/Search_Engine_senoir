           </div> <p>This page is for requesting that a page, file or template be <strong>fully protected</strong>, <strong>create protected</strong> (<a href="/wiki/Wikipedia:Protection_policy#Creation_protection" title="Wikipedia:Protection policy">salted</a>), <strong>extended confirmed protected</strong>, <strong>semi-protected</strong>, added to <strong>pending changes</strong>, <strong>move-protected</strong>, <strong>template protected</strong>, <strong>upload protected</strong> (file-specific), or <strong>unprotected</strong>. Please read the <a href="/wiki/Wikipedia:Protection_policy" title="Wikipedia:Protection policy">protection policy</a> for details. Full protection is used to stop edit warring between multiple users or to prevent vandalism to <a href="/wiki/Wikipedia:High-risk_templates" title="Wikipedia:High-risk templates">high-risk templates</a>; semi-protection and pending changes are usually used only to prevent IP and new user vandalism (see the <a href="/wiki/Wikipedia:Rough_guide_to_semi-protection" title="Wikipedia:Rough guide to semi-protection">rough guide to semi-protection</a>); and move protection is used to stop <a href="/wiki/Wikipedia:Page-move_war" title="Wikipedia:Page-move war">page-move wars</a>. Extended confirmed protection is used where semi-protection has proved insufficient (see the <a href="/wiki/Wikipedia:Rough_guide_to_extended_confirmed_protection" title="Wikipedia:Rough guide to extended confirmed protection">rough guide to extended confirmed protection</a>) </p><p>After a page has been protected, the protection is listed in the page history and logs with a short description indicating why it was protected, and the article is listed on <a href="/wiki/Special:ProtectedPages" title="Special:ProtectedPages">Special:Protectedpages</a>. Further discussion should take place on the Talk page of the article. In the case of full protection due to edit warring, admins do not revert to specific versions of the page, except to get rid of <a href="/wiki/Wikipedia:Vandalism" title="Wikipedia:Vandalism">obvious vandalism</a>. </p>

           <dl>
            <dt>
             <span class="anchor" id="Instructions"></span>Instructions
            </dt> 
            <dd></dd>
           </dl> <p>If you would like to request to have a page protected, have a page's existing protection level lowered or raised, or would like to submit an edit request for a protected page, please follow these steps (note: if you enable the <a href="/wiki/Wikipedia:Twinkle" title="Wikipedia:Twinkle">Twinkle</a> gadget for your Wikipedia account, this process can be largely automated for you): </p> 
           <ol>
            <li>If you are requesting <i>un</i>protection, the correct protocol is to ask the protecting administrator first for unprotection (the admin's user account name can be found in the page's edit history). If for any reason he or she is unresponsive, unwilling, or unable to lower protection then you may list your request on this page.</li> 
            <li>Add a <a href="/wiki/Help:Cheatsheet#Section_headings" title="Help:Cheatsheet">level-3 header</a> (i.e. <code>=== [[Example title]] ===</code>) with the exact title of the page you would like (un)protected. If the page is not in the article namespace, include the namespace prefix (e.g. <code><em>Template</em>:Example template</code> or <code><em>User</em>:Example user</code>). Place your request <big><span style="background:yellow">at the <strong>bottom</strong></span></big> of either the protection or unprotection list, depending on the nature of your request. <em>Please make sure you have spelled the name of the article properly</em>; a common error stems from mistaken requests for the protection of redirects, particularly capitalised ones (<a href="/wiki/Wikipedia:Twinkle" title="Wikipedia:Twinkle">Twinkle</a> can make sure this never happens to you!)</li> 
            <li>Place the pagelinks template below the header, in the form of <code>* {{pagelinks|Example title}}</code> (note the asterisk).</li> 
            <li>If you are requesting protection, write the type of request (full protection, semi-protection, pending changes, move protection) and a brief reason for your request below your header.</li> 
            <li><strong>Please do not add arbitrary requests for a protection expiry time to your request, or request indefinite protection arbitrarily.</strong> 
             <ul>
              <li>If there is a reason for a page to be protected for a certain amount of time, such as protecting a user talk page until the user is unblocked, please make this clear.</li> 
              <li>If you are requesting indefinite semi-protection or pending changes, be aware that it is only applied to articles with <em>endemic and endless vandalism problems</em> that multiple increasing periods of temporary semi-protection or pending changes have failed to stop.</li> 
              <li>Note that different expiry times can be set for edit- and for move-protections— thus, an article can, for example, be semi-edit-protected for a week and also fully-move-protected indefinitely.</li>
             </ul></li> 
            <li>Sign your request with four tildes <code>~~~~</code> and click <code>Publish changes</code>.</li>
           </ol> 
           <ul>
            <li><strong>Note:</strong> editors should <em>not</em> consider requesting page protection as a method for continuing an argument from elsewhere nor as a venue for starting a new discussion regarding article content. If a request contains excessive argument, appears to be intended to resolve a content dispute, includes personal attacks or uncivil comments, or has any other unrelated discussion, it will be removed from this page and no action will be taken.</li>
           </ul>

      <h2><span class="mw-headline" id="Current_requests_for_reduction_in_protection_level">Current requests for reduction in protection level</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Wikipedia:Requests_for_page_protection&amp;action=edit&amp;section=20" title="Edit section: Current requests for reduction in protection level">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r1007196855">
      <div class="module-shortcutanchordiv">
       <span id="WP:RFUP"></span><span id="WP:RFPU"></span>
      </div>
      <div class="module-shortcutboxplain plainlist noprint" role="note">
       <div class="module-shortcutlist">
        <a href="/wiki/Wikipedia:Shortcut" title="Wikipedia:Shortcut">Shortcuts</a>
       </div>
       <ul>
        <li><span class="plainlinks"><a class="external text" href="https://en.wikipedia.org/w/index.php?title=Wikipedia:RFUP&amp;redirect=no">WP:RFUP</a></span></li>
        <li><span class="plainlinks"><a class="external text" href="https://en.wikipedia.org/w/index.php?title=Wikipedia:RFPU&amp;redirect=no">WP:RFPU</a></span></li>
       </ul>
      </div> 
      <p style="text-align:center"><big><span style="color:white;background:red"><i>Before posting, first discuss with the protecting admin at their talk page.</i></span> Only post below if you receive no (favourable) reply.</big></p> 
      <ul>
       <li>To find out the username of the admin who protected the page click on "history" at the top of the page, then click on "View logs for this page" which is under the title of the page. The protecting admin is the username in blue before the words "protected", "changed protection level" or "pending changes". If there are a number of entries on the log page, you might find it easier to select "Protection log" or "Pending changes log" from the dropdown menu in the blue box.</li> 
       <li>Requests to downgrade <a href="/wiki/Wikipedia:Protection_policy#Full_protection" title="Wikipedia:Protection policy">full protection</a> to <a href="/wiki/Wikipedia:Protection_policy#template" title="Wikipedia:Protection policy">template protection</a> on templates and modules can be directed straight here; you do not need to ask the protecting admin first.</li> 
       <li>Requests for removing <a href="/wiki/Wikipedia:Protection_policy#Creation_protection" title="Wikipedia:Protection policy">create protection</a> on <a href="/wiki/Wikipedia:Red_link" title="Wikipedia:Red link">redlinked</a> articles are generally assisted by having a <a href="/wiki/Wikipedia:Drafts" title="Wikipedia:Drafts">draft version</a> of the intended article prepared beforehand.</li> 
       <li>If you want to make spelling corrections or add uncontroversial information to a protected page please add {{<a href="/wiki/Template:Edit_fully-protected" title="Template:Edit fully-protected">Edit fully-protected</a>}} to the article's talk page, along with an explanation of what you want to add to the page. If the talk page is protected please use the section below.</li>
      </ul> 
      <p><i>Check the <a href="/wiki/Wikipedia:Requests_for_page_protection/Rolling_archive" title="Wikipedia:Requests for page protection/Rolling archive">rolling archive</a> if you cannot find your request. Only recently answered requests are still listed here.</i> </p>

      <h2><span class="mw-headline" id="Current_requests_for_edits_to_a_protected_page">Current requests for edits to a protected page</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Wikipedia:Requests_for_page_protection&amp;action=edit&amp;section=22" title="Edit section: Current requests for edits to a protected page">edit</a><span class="mw-editsection-bracket">]</span></span></h2> 
      <link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r1007196855">
      <div class="module-shortcutanchordiv">
       <span id="WP:RFED"></span>
      </div>
      <div class="module-shortcutboxplain plainlist noprint" role="note">
       <div class="module-shortcutlist">
        <a href="/wiki/Wikipedia:Shortcut" title="Wikipedia:Shortcut">Shortcut</a>
       </div>
       <ul>
        <li><span class="plainlinks"><a class="external text" href="https://en.wikipedia.org/w/index.php?title=Wikipedia:RFED&amp;redirect=no">WP:RFED</a></span></li>
       </ul>
      </div> 
      <p><b>Ideally, requests should be made on the article talk page rather than here.</b> </p> 
      <ul>
       <li>Unless the talk page itself is protected, you may instead add the appropriate template among <code>{{<a href="/wiki/Template:Edit_protected" class="mw-redirect" title="Template:Edit protected">Edit protected</a>}}</code>, <code>{{<a href="/wiki/Template:Edit_template-protected" title="Template:Edit template-protected">Edit template-protected</a>}}</code>, <code>{{<a href="/wiki/Template:Edit_extended-protected" title="Template:Edit extended-protected">Edit extended-protected</a>}}</code>, or <code>{{<a href="/wiki/Template:Edit_semi-protected" title="Template:Edit semi-protected">Edit semi-protected</a>}}</code> to the article's talk page if you would like to make a change rather than requesting it here. Doing so will automatically place the page in the appropriate category for the request to be reviewed.</li> 
       <li>Where requests are made due to the editor having a <a href="/wiki/Wikipedia:Conflict_of_interest" title="Wikipedia:Conflict of interest">conflict of interest</a> (COI; see <a href="/wiki/Wikipedia:Suggestions_for_COI_compliance" title="Wikipedia:Suggestions for COI compliance">Wikipedia:Suggestions for COI compliance</a>), the <code>{{<a href="/wiki/Template:Request_edit" title="Template:Request edit">request edit</a>}}</code> template should be used.</li> 
       <li>Requests to move move-protected pages should be made at <a href="/wiki/Wikipedia:Requested_moves" title="Wikipedia:Requested moves">Wikipedia:Requested moves</a>, not here.</li> 
       <li>If the discussion page and the article are <i>both</i> protected preventing you from making an edit request, this page is the right place to make that request. Please see the top of this page for instructions on how to post requests.</li> 
       <li>This page is <i><b>not</b></i> for continuing or starting discussions regarding content should both an article and its discussion page be protected. Please make a request only if you have a specific edit you wish to make.</li>
      </ul> 
      <div role="note" class="hatnote navigation-not-searchable">
       Further information: <a href="/wiki/Wikipedia:Edit_requests" title="Wikipedia:Edit requests">Wikipedia:Edit requests</a>
      </div> 
      <p>Create a level 3 header with a link to the article in question, then a {{pagelinks}} template and then the reason. </p>
