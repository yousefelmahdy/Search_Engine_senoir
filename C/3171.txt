        <h1 class="border-bottom-0">Reverting a commit</h1> 
        <div class="d-none d-lg-block ml-2"> <button class="btn-link Link--secondary js-print tooltipped tooltipped-n" aria-label="Print this article"> <!-- From https://heroicons.dev/ --> 
          <svg fill="none" height="18" width="18" stroke-linecap="round" stroke-linejoin="round" stroke-width="2" viewbox="0 0 24 24" stroke="currentColor">
           <path d="M17 17h2a2 2 0 002-2v-4a2 2 0 00-2-2H5a2 2 0 00-2 2v4a2 2 0 002 2h2m2 4h6a2 2 0 002-2v-4a2 2 0 00-2-2H9a2 2 0 00-2 2v4a2 2 0 002 2zm8-12V5a2 2 0 00-2-2H9a2 2 0 00-2 2v4h10z"></path>
          </svg> </button> 
        </div> 
       </div> 
       <div class="lead-mktg">
        <p>You can revert a specific commit to remove its changes from your branch.</p>
       </div> 
       <nav class="UnderlineNav my-3"> 
        <div class="UnderlineNav-body"> <a href="#" class="UnderlineNav-item platform-switcher" data-platform="mac">Mac</a> <a href="#" class="UnderlineNav-item platform-switcher" data-platform="windows">Windows</a> <a href="#" class="UnderlineNav-item platform-switcher" data-platform="linux">Linux</a> 
        </div> 
       </nav>

      <div class="article-grid-toc border-bottom border-xl-0 pb-4 mb-5 pb-xl-0 mb-xl-0"> 
       <div class="article-grid-toc-content"> 
       </div> 
      </div> 
      <div id="article-contents" class="article-grid-body"> 
       <p>When you revert to a previous commit, the revert is also a commit. The original commit also remains in the repository's history.</p> 
       <div class="extended-markdown tip border rounded-1 mb-4 p-3 color-border-info color-bg-info f5"> 
        <p><strong>Tip:</strong> When you revert multiple commits, it's best to revert in order from newest to oldest. If you revert commits in a different order, you may see merge conflicts.</p> 
       </div> 
       <div class="extended-markdown mac"> 
        <ol> 
         <li>Click <strong>History</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/desktop/history-tab-in-commit-sidebar.png" alt="The history tab selected in the sidebar"></span></li> 
         <li>Right-click the commit you want to revert and click <strong>Revert This Commit</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/desktop/commit-revert-mac.png" alt="The Revert option above the diff view"></span></li> 
        </ol> 
       </div> 
       <div class="extended-markdown windows"> 
        <ol> 
         <li>Click <strong>History</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/desktop/history-tab-in-commit-sidebar.png" alt="The history tab selected in the sidebar"></span></li> 
         <li>Right-click the commit you want to revert and click <strong>Revert This Commit</strong>. <span class="procedural-image-wrapper"><img src="/assets/images/help/desktop/commit-revert-win.png" alt="The Revert option above the diff view"></span></li> 
        </ol>
