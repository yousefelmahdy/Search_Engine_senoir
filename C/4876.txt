        <h3><span class="mw-headline" id="Purpose">Purpose</span></h3> 
        <p>This redirect category (<a href="/wiki/Wikipedia:RCAT" class="mw-redirect" title="Wikipedia:RCAT">rcat</a>) template sorts redirect pages into one of two categories: </p> 
        <ol>
         <li><a href="/wiki/Category:Redirects_to_project_space" title="Category:Redirects to project space">Category:Redirects to project space</a> – for <a href="/wiki/Wikipedia:PNR" class="mw-redirect" title="Wikipedia:PNR">pseudo-namespace redirects (PNRs)</a> (for example, MOS:) and <a href="/wiki/Wikipedia:CNR" class="mw-redirect" title="Wikipedia:CNR">cross-namespace redirects (CNRs)</a>, or</li> 
         <li><a href="/wiki/Category:Redirects_to_Wikipedia_project_pages" title="Category:Redirects to Wikipedia project pages">Category:Redirects to Wikipedia project pages</a> – for <a href="/wiki/Wikipedia:Project_namespace" title="Wikipedia:Project namespace">project namespace</a> redirects (for example, Wikipedia: or its shortcut, WP:) to other project pages</li>
        </ol> 
        <p>This rcat template also populates the following: </p> 
        <dl>
         <dd>
          &nbsp;3. <a href="/wiki/Category:Unprintworthy_redirects" title="Category:Unprintworthy redirects">Category:Unprintworthy redirects</a> – only redirects from <a href="/wiki/Wikipedia:Main_namespace" class="mw-redirect" title="Wikipedia:Main namespace">main-article namespace (mainspace)</a> 
          <ul>
           <li>In 2003, efforts were begun to support the <a href="/wiki/Wikimedia_Foundation" title="Wikimedia Foundation">Wikimedia Foundation</a>'s goal of increasing access and availability of <i>Wikipedia</i> articles in <span class="plainlinks"><a class="external text" href="https://en.wikipedia.org/w/index.php?title=Wikipedia:Pushing_to_1.0&amp;oldid=1319195">printed versions</a></span>. Mainspace redirects that target pages in the Wikipedia project namespace are not suitable for a printed version. In other words, they are <a href="/wiki/Wikipedia:Printworthiness" class="mw-redirect" title="Wikipedia:Printworthiness">unprintworthy</a>.</li>
          </ul>
         </dd>
        </dl>

        <h3><span class="mw-headline" id="Usage">Usage</span></h3> 
        <div role="note" class="hatnote navigation-not-searchable">
         Use this rcat template in <b>any <a href="/wiki/Wikipedia:Namespace" title="Wikipedia:Namespace">namespace</a> including <a href="/wiki/Wikipedia:Project_namespace" title="Wikipedia:Project namespace">project (Wikipedia)</a> namespace</b>.
        </div> 
        <ul>
         <li>Apply this rcat template to a redirect page by using the following <a href="/wiki/Wikitext" class="mw-redirect" title="Wikitext">wikitext</a>:</li>
        </ul> 
        <pre>#REDIRECT [[Wikipedia:(target page name)]]

{{Rcat shell|
{{R to project namespace}}
}}
</pre> 
        <dl>
         <dd>
          and <i>when the redirect is in article mainspace</i>, printworthiness should also be tagged:
         </dd>
        </dl> 
        <pre>#REDIRECT [[Wikipedia:(target page name)]]

{{Rcat shell|
{{R to project namespace}}
{{R unprintworthy}}
}}
</pre> 
        <dl>
         <dd>
          Template {{<a href="/wiki/Template:Rcat_shell" class="mw-redirect" title="Template:Rcat shell">Rcat shell</a>}} is an alias for the <i>Redirect category shell</i> template, which may be used to add as many appropriate rcat templates and their parameters as needed to a redirect, usually from one to seven. For more information, see the documentation on its template page. This rcat may also tag a redirect individually:
         </dd>
        </dl> 
        <pre>#REDIRECT [[Wikipedia:(target page name)]]

{{R to project namespace}}
</pre> 
        <dl>
         <dd>
          This is in accord with instructions found at <a href="/wiki/Wikipedia:REDCAT" class="mw-redirect" title="Wikipedia:REDCAT">Wikipedia:REDCAT</a>.
         </dd>
        </dl> 
        <ul>
         <li>Use this rcat template to tag any redirect page to a page in the <a href="/wiki/Wikipedia:Project_namespace" title="Wikipedia:Project namespace">project (Wikipedia: or WP:) namespace</a>, whether the redirect page itself is in project namespace or in any other namespace. These redirect pages will automatically be sorted as either inside or outside of project namespace.</li>
        </ul>
