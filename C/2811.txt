       <h3 id="about-metrics-and-reports"><a href="#about-metrics-and-reports">About metrics and reports</a></h3> 
       <p>Data available in GitHub Insights are divided into key metrics and reports. For more information about available metrics, see "<a href="/en/enterprise-server@3.1/insights/exploring-your-usage-of-github-enterprise/metrics-available-with-github-insights">Metrics available with GitHub Insights</a>."</p> 
       <p>You can view goals and success rates for each key metric. For more information, see "<a href="/en/enterprise-server@3.1/insights/installing-and-configuring-github-insights/managing-goals">Managing goals</a>"</p> 
       <p>You will only have access to the GitHub Insights data which you also have access to in GitHub Enterprise.</p> 
       <p>You can filter the data included in key metrics or reports by teams, repositories, or date range.</p>
