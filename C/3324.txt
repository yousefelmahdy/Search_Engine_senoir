       <p><a href="/wiki/MediaWiki:Spam-blacklist" title="MediaWiki:Spam-blacklist">Mediawiki:Spam-blacklist</a> is meant to be used by the <a href="https://www.mediawiki.org/wiki/Extension:SpamBlacklist" class="extiw" title="mw:Extension:SpamBlacklist">spam blacklist extension</a>. Unlike the <a href="https://meta.wikimedia.org/wiki/Spam_blacklist" class="extiw" title="m:Spam blacklist">meta spam blacklist</a>, this blacklist affects pages on the English Wikipedia only. Any <a href="/wiki/Wikipedia:Administrators" title="Wikipedia:Administrators">administrator</a> may edit the spam blacklist. See <a href="/wiki/Wikipedia:Spam_blacklist" title="Wikipedia:Spam blacklist">Wikipedia:Spam blacklist</a> for more information about the spam blacklist. </p> 
       <hr> 
       <div style="text-align: center;">
        <big><u><b>Instructions for editors</b></u></big>
       </div>
       <br> 
       <p>There are 4 sections for posting comments below. Please make comments in the appropriate section. These links take you to the appropriate section: </p> 
       <ol>
        <li><i><a href="#Proposed_additions">Proposed additions</a></i></li> 
        <li><i><a href="#Proposed_removals">Proposed removals</a></i></li> 
        <li><i><a href="#Troubleshooting_and_problems">Troubleshooting and problems</a></i></li> 
        <li><i><a href="#Discussion">Discussion</a></i></li>
       </ol> 
       <p>Each section has a message box with instructions. In addition, please <b>sign your posts</b> with ~~~~ after your comment. </p>
       <p>Completed requests are <a href="/wiki/MediaWiki_talk:Spam-blacklist/archives" title="MediaWiki talk:Spam-blacklist/archives">archived</a>. All additions and removals are also <a href="/wiki/MediaWiki_talk:Spam-blacklist/log" title="MediaWiki talk:Spam-blacklist/log">logged</a>. </p> 
       <hr> 
       <div style="text-align: center;"> <big><u><b>Instructions for admins</b></u></big>
       </div>
       <br> 
       <div style="text-align: center;">
        <i><b>Any admin unfamiliar with this page should probably <a href="/wiki/Wikipedia:Spam-blacklisting" title="Wikipedia:Spam-blacklisting">read this first</a>, thanks.<br> If in doubt, please leave a request and a spam-knowledgeable admin will follow-up.</b></i>
       </div>
       <br> 
       <ol>
        <li>Does the site have any validity to the project?</li> 
        <li>Have links been placed after warnings/blocks? Have other methods of control been exhausted? Would referring this to our anti-spam bot, <a href="/wiki/User_talk:XLinkBot/RevertList" title="User talk:XLinkBot/RevertList">XLinkBot</a> be a more appropriate step? Is there a <a href="/wiki/Wikipedia_talk:WikiProject_Spam" title="Wikipedia talk:WikiProject Spam">WikiProject Spam report</a>? If so, a permanent link would be helpful.</li> 
        <li>Please ensure <b>all links have been removed</b> from articles and discussion pages <b>before blacklisting</b>. (They do not have to be removed from user or user talk pages).</li> 
        <li>Make the entry at the bottom of the list (before the last line). <b>Please do not do this unless you are familiar with <a href="/wiki/Regular_expressions" class="mw-redirect" title="Regular expressions">regex</a> — the disruption that can be caused is substantial</b>.</li> 
        <li>Close the request entry on here using either {{<a href="/wiki/Template:Done" title="Template:Done">done</a>}} or {{<a href="/wiki/Template:Not_done" title="Template:Not done">not done</a>}} as appropriate. The request should be left open for a week maybe as there will often be further related sites or an appeal in that time.</li> 
        <li><b>Log the entry</b>. <span style="color:red;"><big><b>Warning:</b></big></span> <b>if you do not log any entry you make on the blacklist, it may well be removed if someone appeals and no valid reasons can be found</b>. To log the entry, you will need this number - 1027052457 <b>after</b> you have closed the request. See <a href="/wiki/MediaWiki_talk:Spam-blacklist/log#Adding_entries" title="MediaWiki talk:Spam-blacklist/log">here</a> for more info on logging.</li>
       </ol>

           <b>Instructions for proposed additions</b>
          </div> 
          <ol>
           <li>Please add new entries to the <b>bottom</b> of this section.</li> 
           <li>Please only use the basic URL – <b>example.com</b> , not <a rel="nofollow" class="external free" href="https://www.example.com">https://www.example.com</a>.</li> 
           <li>Consider informing editors whose actions are discussed here.</li>

             </dd>
            </dl>
           </dd>

              <samp>{{User summary|Jimbo Wales}}</samp> <small><i>-- do not use "subst:" with this template</i></small>
             </dd>
            </dl>
           </dd> 
           <dd>
            <b>{{<a href="/wiki/Template:Link_summary" title="Template:Link summary">Link summary</a>}}</b> – to report <i><b>spam domains</b></i>: 
            <dl>
             <dd>
              <samp><b>{{Link summary|example.com}}</b></samp> <small><i>-- do not use "subst:" with this template</i></small>
             </dd> 
             <dd>
              <small><i>Do not include the "http[s]://www." portion of an <a href="/wiki/URL" title="URL">URL</a> inside this template, nor anything behind the domain name. Including this template will give tools to investigate the domain, and will result in COIBot refreshing the link-report. ('<a href="/wiki/Wikipedia:WikiProject_Spam/LinkReports/example.com" title="Wikipedia:WikiProject Spam/LinkReports/example.com">COIBot</a>')</i></small>

            <b>{{<a href="/wiki/Template:BLRequestLink" title="Template:BLRequestLink">BLRequestLink</a>}}</b> - to suggest specific links to be blacklisted

      <h2><span class="mw-headline" id="Addition_to_the_COIBot_reports">Addition to the COIBot reports</span></h2> 
      <p>The lower list in the COIBot reports now have after each link four numbers between brackets (e.g. "www.example.com (0, 0, 0, 0)"): </p> 
      <ol>
       <li>first number, how many links did <i>this</i> user add (is the same after each link)</li> 
       <li>second number, how many times did <i>this</i> link get added to wikipedia (for as far as the linkwatcher database goes back)</li> 
       <li>third number, how many times did <i>this</i> user add <i>this</i> link</li> 
       <li>fourth number, to how many different wikipedia did <i>this</i> user add <i>this</i> link.</li>
      </ol> 
      <p>If the third number or the fourth number are high with respect to the first or the second, then that means that the user has at least a preference for using that link. Be careful with other statistics from these numbers (e.g. good user who adds a lot of links). If there are more statistics that would be useful, please notify me, and I will have a look if I can get the info out of the database and report it. This data is available in real-time on IRC. <br> </p>
