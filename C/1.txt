     <h1><span class="color_h1">onkeypress</span> Event</h1> 
     <p> <a href="dom_obj_event.asp" class="w3-btn w3-white w3-border" title="Events">? DOM Events</a> <a href="obj_keyboardevent.asp" class="w3-btn w3-white w3-border" title="KeyboardEvent">? KeyboardEvent</a> </p> 
     <div class="w3-example"> 
      <h3>Example</h3> 
      <p>Execute a JavaScript when a user presses a key:</p> 
      <div class="w3-code notranslate htmlHigh">
        &lt;input type="text" onkeypress="myFunction()"&gt;
      </div> <a target="_blank" href="tryit.asp?filename=tryjsref_onkeypress" class="w3-btn w3-margin-bottom">Try it Yourself »</a> 
     </div> 
     <hr> 
     <h2>Definition and Usage</h2> 
     <p>The onkeypress event occurs when the user presses a key (on the keyboard).</p> 
     <p><b>Tip:</b> The order of events related to the onkeypress event:</p> 
     <ol> 
      <li><a href="event_onkeydown.asp">onkeydown</a></li> 
      <li>onkeypress</li> 
      <li><a href="event_onkeyup.asp">onkeyup</a></li> 
     </ol> 
     <p><b>Note:</b> The onkeypress event is not fired for all keys (e.g. ALT, CTRL, SHIFT, ESC) in all browsers. To detect only whether the user has pressed a key, use the <a href="event_onkeydown.asp">onkeydown</a> event instead, because it works for all keys.</p>
