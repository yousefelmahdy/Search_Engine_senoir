      <p><b>WikiProject Categories</b> has been set up to help manage the categories of Wikipedia and improve them. The goal is to categorize all articles in a way that is consistent with the guidelines at <a href="/wiki/Wikipedia:Categorization" title="Wikipedia:Categorization">Wikipedia:Categorization</a>. </p>
      <p>Discussions about categorization policy happen at <a href="/wiki/Wikipedia_talk:Categorization" title="Wikipedia talk:Categorization">Wikipedia talk:Categorization</a>. That page is also the place for solving specific issues related to category organization. </p>
      <p>To discuss deleting, merging, or renaming categories, see <a href="/wiki/Wikipedia:Categories_for_discussion" title="Wikipedia:Categories for discussion">Wikipedia:Categories for discussion</a>. </p>
      <p>Please direct all talk to <a href="/wiki/Wikipedia_talk:WikiProject_Categories" title="Wikipedia talk:WikiProject Categories">Wikipedia talk:WikiProject Categories</a> if you would like to help or comment on the project. </p>
      <p>There is a <a href="/wiki/Wikipedia:WikiProject_Categories/uncategorized" title="Wikipedia:WikiProject Categories/uncategorized">task force</a> that puts uncategorized articles into categories. </p>
      <p>You may wish to view <a href="/wiki/Special:PrefixIndex/Wikipedia:WikiProject_Categories/" title="Special:PrefixIndex/Wikipedia:WikiProject Categories/">all subpages of this page</a>. </p>
