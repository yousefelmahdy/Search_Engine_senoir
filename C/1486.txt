       <h2><span class="mw-headline" id="Usage">Usage</span></h2> 
       <div class="templatebox-usagesample"> 
        <p>{{Multilingual description |<var>xx</var>= }} </p> 
       </div> 
       <p>Or use the shorthand <tt>{{Mld|… parameters as described …}}</tt> </p>
       <p><small></small> </p> 
       <div class="templatebox-languagedependant"> 
        <p>For the creation of the form-select and folding of languages, JavaScript is required. This JavaScript is implemented as default gadget at Wikimedia Commons, namely “<i>Language select</i>”, and can be switched off <a href="/wiki/Special:Preferences#mw-prefsection-gadgets" title="Special:Preferences">amending the preferences</a>. <small>(<span class="plainlinks"><a class="external text" href="https://commons.wikimedia.org/w/index.php?title=Template:Multilingual_description&amp;withJS=MediaWiki:ActivateGadget.js&amp;gadgetname=LanguageSelect">Activate the LanguageSelect gadget now!</a></span> // <span class="plainlinks"><a class="external text" href="https://commons.wikimedia.org/w/index.php?title=Template:Multilingual_description&amp;withJS=MediaWiki:ActivateGadget.js&amp;disablegadget=1&amp;gadgetname=LanguageSelect">Disable the LanguageSelect gadget now!</a></span>)</small> </p>
        <p>The gadget will also fold descriptions in multiple languages and create a form-select, if more than a threshold are written consecutively on a page, using language templates like <a href="/wiki/Template:En" title="Template:En"><span style="font-family:monospace,monospace;">{{En}}</span></a>, <a href="/wiki/Template:Fr" title="Template:Fr"><span style="font-family:monospace,monospace;">{{Fr}}</span></a> and <a href="/wiki/Template:Es" title="Template:Es"><span style="font-family:monospace,monospace;">{{Es}}</span></a>. However, if the text and the corresponding language code to display (for example, "en") is not listed in the template, all of the other languages listed in the template will be displayed instead, by default. </p>
