      <h2><span class="mw-headline" id="About_the_help_system">About the help system</span></h2> 
      <div role="note" class="hatnote navigation-not-searchable">
       See also: <a href="/wiki/Wikipedia:Project_namespace#Wikipedia_how-to_and_information_pages" title="Wikipedia:Project namespace">Wikipedia:Project namespace §&nbsp;Wikipedia how-to and information pages</a>
      </div> 
      <div class="floatright">
       <a href="/wiki/Wikipedia:Help_desk" title="Wikipedia:Help desk"><img alt="Wiki-help.png" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e1/Wiki-help.png/220px-Wiki-help.png" decoding="async" width="220" height="146" srcset="//upload.wikimedia.org/wikipedia/commons/e/e1/Wiki-help.png 1.5x" data-file-width="300" data-file-height="199"></a>
      </div> 
      <p>The help resource system of Wikipedia consists of a wide variety of <a href="/wiki/Wikipedia:HOWTOPAGES" class="mw-redirect" title="Wikipedia:HOWTOPAGES">how-to and information pages</a>, written to differing levels of technical expertise and in widely varying styles. This variety however has its flaws: it is easy to navigate to a page that regurgitates the same information, contains concepts that have not already been covered, is overly complicated for one's needs, or pages with a listing of links to other pages with listings of links. Information is also spread over a variety of namespace prefixes. The <i><a href="/wiki/Wikipedia:Wikipedia_namespace" class="mw-redirect" title="Wikipedia:Wikipedia namespace">Wikipedia namespace</a></i> (prefix <i>Wikipedia:</i>): contains many types of pages connected with the Wikipedia project itself: basic information (e.g., <a href="/wiki/Wikipedia:Administration" title="Wikipedia:Administration">Wikipedia:Administration</a>), policies (e.g., <a href="/wiki/Wikipedia:Editing_policy" title="Wikipedia:Editing policy">Wikipedia:Editing policy</a>), guidelines (e.g., <a href="/wiki/Wikipedia:Manual_of_Style" title="Wikipedia:Manual of Style">Wikipedia:Manual of Style</a>), essays (e.g., <a href="/wiki/Wikipedia:Reasonability_rule" title="Wikipedia:Reasonability rule">Wikipedia:Reasonability rule</a>), how-to pages (e.g., <a href="/wiki/Wikipedia:Purging" class="mw-redirect" title="Wikipedia:Purging">Wikipedia:Purging</a>). The <i><a href="/wiki/Wikipedia:Help_namespace" title="Wikipedia:Help namespace">Help namespace</a></i> (prefix <i>Help:</i>): contains pages which provide help in using Wikipedia and its software, both for users of the encyclopedia and for editors (e.g., <a href="/wiki/Help:Footnotes" title="Help:Footnotes">Help:Footnotes</a>). However the <i>Wikipedia namespace</i> also contains many help style pages that in some cases contain repeated information or more detailed information. For example, <a href="/wiki/Wikipedia:Media_help" class="mw-redirect" title="Wikipedia:Media help">Wikipedia:Media help</a> and <a href="/wiki/Help:Files" title="Help:Files">Help:Files</a> are help pages located in different prefixes covering some of the same information presented in differing levels of technical expertise. There are also pages that have completely different styles of presenting the same information (e.g., <a href="/wiki/Help:Referencing_for_beginners" title="Help:Referencing for beginners">Help:Referencing for beginners</a> and <a href="/wiki/Help:Introduction_to_referencing" class="mw-redirect" title="Help:Introduction to referencing">Help:Introduction to referencing</a>). For this reason <a href="/wiki/Help:Redirect" title="Help:Redirect">redirects</a> and <a href="/wiki/Wikipedia:Hatnote" title="Wikipedia:Hatnote">hatnotes</a> are often set up between these two namespaces. Aside from the static pages there are a multitude of interactive assistances pages see, <a href="/wiki/Wikipedia:Requests" class="mw-redirect" title="Wikipedia:Requests">Wikipedia:Requests</a> for a listing of those. </p> 
      <h2><span class="mw-headline" id="How_to_utilize_help">How to utilize help</span></h2> 
      <h3><span class="mw-headline" id="Where_to_start_for_new_editors">Where to start for new editors</span></h3> 
      <p>If you would like to get a good overview of the whole project and how it works, you could read through the introductory page at <a href="/wiki/Wikipedia:Contributing_to_Wikipedia" title="Wikipedia:Contributing to Wikipedia">Wikipedia:Contributing to Wikipedia</a> or go through the tutorial at <a href="/wiki/Help:Introduction" title="Help:Introduction">Help:Introduction</a>. <a href="/wiki/Help:Getting_started" title="Help:Getting started">Help:Getting started</a> contains a listing of introductions and tutorials on the various aspects of Wikipedia. </p>
      <p>If you just want help on how to read Wikipedia, see <a href="/wiki/Wikipedia:FAQ/Readers" title="Wikipedia:FAQ/Readers">Wikipedia:FAQ/Readers</a>. For information on searching and browsing the encyclopedia, see <a href="/wiki/Help:Navigation" title="Help:Navigation">Help:Navigation</a>. </p> 
      <h3><span class="mw-headline" id="Searching_help_pages">Searching help pages</span></h3> 
      <p>If you type keywords of your question into the search box, a page relevant to your question will usually turn up in the search results, e.g. 'How do I create an article?' ( type in 'create an article' ), 'What size are thumbnails?' ( type in 'thumbnail size' ). The search needs to be limited to the help pages using the advanced search page (which appears after you have clicked 'search') or you can use the search box in <a href="/wiki/Help:Contents" title="Help:Contents">Help:contents</a> (accessed at any time by clicking <i>Help</i> displayed on the left side of all pages.). This is duplicated below. </p>

      <h3><span class="mw-headline" id="Interactive_assistances">Interactive assistances</span></h3> 
      <ul>
       <li>You can ask for help on using Wikipedia at the <a href="/wiki/Wikipedia:Help_desk" title="Wikipedia:Help desk">help desk</a> and <a href="/wiki/Wikipedia:Teahouse" title="Wikipedia:Teahouse">teahouse</a>.</li>

      <h2><span class="mw-headline" id="Contributing_to_the_help_pages">Contributing to the help pages</span></h2> 
      <p>Like articles, most of the help pages can be edited - so feel free to improve any pages you come across. When a high traffic page has been protected to help prevent vandalism, suggestions can still be made on the talk page. Some coordination happens at <a href="/wiki/Wikipedia:Help_Project" title="Wikipedia:Help Project">Wikipedia:Help Project</a>. </p>
