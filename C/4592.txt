      <p><a href="/wiki/Wikipedia:Shortcut" title="Wikipedia:Shortcut">Shortcuts</a> are historically reserved for Wikipedia project reference pages and are now used in and for all namespaces. They are often referenced in edit history. Please note that this category is not meant for shorter, abbreviated titles of articles in <a href="/wiki/Wikipedia:Mainspace" class="mw-redirect" title="Wikipedia:Mainspace">mainspace</a>. Please use the templates {{<a href="/wiki/Template:R_from_initialism" title="Template:R from initialism">R from initialism</a>}} and {{<a href="/wiki/Template:R_from_acronym" title="Template:R from acronym">R from acronym</a>}} for such redirects to articles in the main <a href="/wiki/Wikipedia:Namespace" title="Wikipedia:Namespace">namespace</a>. Please take note that <a href="/wiki/Category:Redirects_from_template_shortcuts" title="Category:Redirects from template shortcuts">Category:Redirects from template shortcuts</a> is a subcategory of this category. See also {{<a href="/wiki/Template:R_from_template_shortcut" class="mw-redirect" title="Template:R from template shortcut">R from template shortcut</a>}} for more information. </p>
