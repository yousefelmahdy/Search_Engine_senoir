      <div id="article-contents" class="article-grid-body"> 
       <p>The secret scanning API lets you retrieve and update secret scanning alerts from a private repository. For more information on secret scanning, see "<a href="/en/code-security/secret-security/about-secret-scanning">About secret scanning</a>."</p> 
       <div> 
        <div> 
         <h3 id="list-secret-scanning-alerts-for-a-repository" class="pt-3"> <a href="#list-secret-scanning-alerts-for-a-repository">List secret scanning alerts for a repository</a> </h3> 
         <p>Lists all secret scanning alerts for a private repository, from newest to oldest. To use this endpoint, you must be an administrator for the repository or organization, and you must use an access token with the <code>repo</code> scope or <code>security_events</code> scope.</p> 
         <p>GitHub Apps must have the <code>secret_scanning_alerts</code> read permission to use this endpoint.</p> 
        </div> 
        <pre><code><span class="color-bg-info-inverse color-text-inverse rounded-1 px-2 py-1" style="text-transform: uppercase">get</span> /repos/{owner}/{repo}/secret-scanning/alerts</code></pre>
