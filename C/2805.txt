      <div id="article-contents" class="article-grid-body"> 
       <p>The GitHub Discussions GraphQL API allows you to get, create, edit, and delete discussion posts. For more information about GitHub Discussions, see "<a href="/en/discussions/collaborating-with-your-community-using-discussions/about-discussions">?About discussions</a>."</p> 
       <p>This API is available for authenticated users, OAuth Apps, and GitHub Apps. Access tokens require the <code>repo</code> scope for private repositories and the <code>public_repo</code> scope for public repositories. For more information, see "<a href="/en/developers/apps/scopes-for-oauth-apps">Scopes for OAuth apps</a>."</p> 
       <p>To use this API, you must include <code>GraphQL-Features: discussions_api</code> in the <code>HTTP</code> header.</p>

       <h2 id="mutations"><a href="#mutations">Mutations</a></h2> 
       <p>These mutations follow the same implementation pattern that other mutations in the GraphQL API. Each mutation accepts a single argument of an <code>Input</code> type, named after the mutation, and returns a <code>Payload</code> type containing the fields specified.</p> 
       <p>For example, this is a basic <code>createDiscussion</code> mutation that will create a new discussion:</p> 
       <pre><code class="hljs language-graphql"><span class="hljs-keyword">mutation</span> {
  <span class="hljs-comment"># input type: CreateDiscussionInput</span>
  createDiscussion(<span class="hljs-keyword">input</span>: {repositoryId: <span class="hljs-string">"1234"</span>, categoryId: <span class="hljs-string">"5678"</span>, body: <span class="hljs-string">"The body"</span>, title: <span class="hljs-string">"The title"</span>}) {

    <span class="hljs-comment"># response type: CreateDiscussionPayload</span>
    discussion {
      id
    }
  }
}
</code></pre> 
       <h3 id="creatediscussion"><a href="#creatediscussion">createDiscussion</a></h3> 
       <p>Input fields:</p> 
       <ul> 
        <li><code>body: String!</code> The body of the new discussion.</li> 
        <li><code>title: String!</code> The title of the new discussion.</li> 
        <li><code>repositoryId: ID!</code> The ID of a repository in which to create the discussion.</li> 
        <li><code>categoryId: ID!</code> The ID of a <code>DiscussionCategory</code> within this repository.</li> 
        <li><code>clientMutationId: String</code> A unique identifier for the client performing the mutation.</li> 
       </ul> 
       <p>Return type fields:</p> 
       <ul> 
        <li><code>clientMutationId: String</code> The unique identifier provided as an input.</li> 
        <li><code>discussion: Discussion</code> The discussion that was created.</li> 
       </ul>
